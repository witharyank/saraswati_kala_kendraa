# Saraswati Kala Kendra

Official website for Saraswati Kala Kendra, built with Next.js.

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- Next.js
- TypeScript
- CSS

## Build

```bash
npm run build
```
